## 🎨 CÓDIGO DE COLORES

- **AZUL (`#00d9ff`)**: Todo lo relacionado con TECLADO
  - Bordes de sección
  - Títulos
  - Badges
  - Registros
  - Eventos en tablas

- **VERDE (`#00ff88`)**: Todo lo relacionado con MOUSE
  - Bordes de sección
  - Títulos
  - Badges
  - Registros
  - Eventos en tablas

## 🔑 ELEMENTOS CLAVE

### Para Teclado:
1. **Buffer de Texto**: Muestra exactamente lo que escribes
2. **4 Registros**: Data, Status, Command, Control
3. **Tabla de Eventos**: Solo teclas presionadas/soltadas
4. **Flag IRQ1**: Se ilumina al presionar teclas

### Para Mouse:
1. **Estadísticas**: Posición X/Y, movimientos, clicks
2. **4 Registros**: Data, Status, Command, Control
3. **Tabla de Eventos**: Solo movimientos y clicks del mouse
4. **Flag IRQ12**: Se ilumina al mover/clickear

## 📱 NAVEGACIÓN

```
Scroll ↓
  ├─ Ver controlador de TECLADO completo
  │  └─ Presiona teclas → Ve registros y eventos
  │
  ├─ Scroll más ↓
  │
  └─ Ver controlador de MOUSE completo
     └─ Mueve mouse → Ve registros y eventos
```

## ✅ VENTAJAS

1. **Separación Clara**: No hay confusión entre dispositivos
2. **Colores Consistentes**: Azul = teclado, Verde = mouse
3. **Información Contextual**: Todo relacionado está junto
4. **Educativo**: Fácil entender cómo funciona cada controlador
5. **Escalable**: Fácil agregar más controladores en el futuro
